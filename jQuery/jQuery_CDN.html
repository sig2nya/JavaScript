jQuery Basic Syntax : $(selector).action()
1. $ Sign to define & access jQuery
2. Selector to "query" HTML Elements
3. jQuery action() to be performed on the element

ex) $(this).hide() - hides the current element
$("p").hide() - hides all p elements
$(".test").hide() - hides all elements class = test
$("#test").hide() - hides all elements id = test
